### Getting Started


Go to this <a href="https://github.com/maizzle/maizzle">repo<a>


Clone this starter:

```
npx degit maizzle/maizzle my-project
```

Install dependencies:

```
cd my-project

npm install
```

Start local development:

```
npm run dev
```

Build emails for production:

```
npm run build
```
